# ClickStock

Sistema de Gestión de Stock desarrollado en Python utilizando wxPython para la interfaz gráfica y SQLite como base de datos.

## Integrantes

- Iván Fernando Rodríguez

## Tecnologías utilizadas

- Python 3.13 (Recomendado)
- wxPython 4.2.3
- SQLite

## Instalación

1. Clonar el repositorio:

bash
git clone https://github.com/Panconmatecocido/ClickStock.git


2. Instalar las dependencias:

bash
pip install -r requirements.txt


## Ejecución

Desde la carpeta del proyecto ejecutar:

bash
python main.py


## Funcionalidades

- Gestión de categorías.
- Gestión de productos.
- Control de stock.
- Reportes.
- Creditos.
- Splash Screen.
